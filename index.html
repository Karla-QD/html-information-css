Introducción

HTML (Hypertext Markup Language) es el lenguaje de marcado estándar utilizado para crear páginas web. Es la base de la mayoría de los sitios en la web y se utiliza para estructurar el contenido y dar formato al texto, imágenes, videos y otros elementos que se encuentran en una página web.

Con HTML, se puede crear una página web desde cero o modificar una existente para que sea más atractiva y fácil de navegar. Además, HTML es un lenguaje fácil de aprender y se puede utilizar en combinación con otros lenguajes de programación para agregar funcionalidad adicional a una página web.

¿Que debería saber?
  • Todo documento HTML comienza con una declaración DOCTYPE, seguida del elemento HTML principal, que contiene dos elementos secundarios:HEAD Y BODY.
  • Las etiquetas tienen atributos,algunos de los atributos más comunes incluyen class, id, href, src, alt y style.
  • Los navegadores interpretan el código HTML de manera diferente, lo que puede afectar la apariencia y funcionalidad de una página web.
¿Que contiene el elemento HEAD?

La sección HEAD en un documento HTML contiene información sobre el documento que no se muestra en la página web en sí misma, sino que se utiliza para proporcionar información a los navegadores y motores de búsqueda. Algunos de los elementos comunes que se pueden encontrar en la sección HEAD incluyen:

  • La etiqueta TITLE, que define el título de la página web que se muestra en la pestaña del navegador.
  • Metaetiquetas META, que proporcionan información adicional sobre el documento, como descripciones, palabras clave, codificación de caracteres y autor.
  • Enlaces a hojas de estilo css,LINK que se utilizan para definir la apariencia de la página web.
¿Que contiene el elemento BODY?

La sección BODY en un documento HTML contiene el contenido visible de una página web, que incluye texto, imágenes, videos, enlaces y otros elementos multimedia. Los elementos que se pueden incluir en la sección BODY se definen mediante etiquetas HTML y se organizan en una estructura jerárquica que determina cómo se visualiza el contenido. Algunos de los elementos más comunes que se pueden encontrar en la sección BODY incluyen:

  • Encabezados (h1, h2, h3, etc.), que se utilizan para definir los títulos y subtítulos de una página web..
  • Párrafos (p), que se utilizan para dividir el texto en secciones legibles y lógicas.
  • Imágenes (img), que se utilizan para agregar gráficos y otros elementos visuales a una página web.
  • Listas (ul, ol), que se utilizan para organizar elementos en una lista con viñetas o números.
  • Tablas (table) , que se utilizan para organizar datos en filas y columnas.
  • Enlaces (a) , que se utilizan para crear enlaces a otras páginas web o recursos en línea.
  • Formularios (form), que se utilizan para recopilar información del usuario y enviarla a un servidor.

¿Como mostrar codigo en html?

Con la etiqueta (code), se puede mostrar fragmentos de codigo, algunos ejemplos:

Suma dos numeros

function suma(a, b) { return a + b; }

Factorial

def factorial(n): if n == 0: return 1 else: return n * factorial(n-1)

Hello world!

function getHelloWorldString() { return "Hello World!"; }

Reverse string

def reverse_string(str) reversed = "" i = str.length - 1 while i >= 0 reversed += str[i] i -= 1 end return reversed end

Fibonacci

public static int Fibonacci(int n) { if (n == 0 || n == 1) { return n; } else { return Fibonacci(n - 1) + Fibonacci(n - 2); } }
